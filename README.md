# 🚗 BMW Sales Analytics Dashboard

## 📌 Project Overview
This repository features an interactive **BMW Sales Analytics Dashboard** built using **Power BI**. The project focuses on converting raw automotive sales data into strategic business intelligence, allowing stakeholders to track revenue health, analyze market demand, and evaluate product-level performance across different dimensions.

---

## 🎯 Project Objectives & Business Value
This dashboard serves as a strategic tool to help businesses:
* **Optimize Revenue Tracking:** Monitor financial health, sales performance, and period-over-period growth.
* **Evaluate Product Demand:** Identify the best-selling BMW models and vehicle categories to optimize inventory.
* **Geographical Insights:** Track regional market shares and identify high-performing locations.
* **Understand Consumer Behavior:** Analyze customer purchasing patterns and sales segmentation for targeted marketing.
* **Data-Driven Decisions:** Enable stakeholders to make fast, informed choices through key performance indicators (KPIs).

---

## 📊 Core Dashboard Features
* **Sales Performance Analytics:** High-level executive views featuring dynamic KPI cards for revenue, volume, and growth trends over time.
* **Product & Model Insights:** In-depth breakdown of vehicle sales distribution, allowing direct comparison between different BMW models.
* **Regional & Market Comparison:** Interactive geographical visuals that highlight sales concentration and market trends across locations.
* **Advanced Interactivity:** Fully dynamic reporting utilizing cross-filtering, custom slicers, and interactive tooltips for seamless data exploration.

---

## 🛠️ Technical Stack & Skills Demonstrated
By developing this project, I applied end-to-end Business Intelligence (BI) methodologies:
* **Data Extraction & ETL:** Used **Power Query** for data cleaning, handling missing values, and transforming raw data into an analytical-ready state.
* **Data Modeling:** Designed efficient star/snowflake schemas to establish optimized relationships between tables.
* **Analytical Calculations:** Authored custom **DAX (Data Analysis Expressions)** measures for advanced metrics, time-intelligence analysis, and KPI tracking.
* **Data Storytelling:** Designed an intuitive, clean, and professional user interface following modern dashboard design principles.
